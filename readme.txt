== Installation ==
Lade die zip datei in dein Pluginsverzeichnis (wp-content/plugins/) und entpacke es. 
Aktiviere das Plugin im Plugins Menue und fuege das Widget im entsprechenden Menue zu
Deiner Seitenleiste hinzu.

== Description ==

Update 5.06.11 
Icons werden wieder angezeigt, optional können Temperatur-Tagesmaximum -Tagesminimum ausgegeben werden. 
Das Widget zeigt das Wetter in Deiner Stadt an; geb einfach deine Postleitzahl ein und 
fuer wieviele Tage Du eine Prognose moechtest. Neu hinzugekommen ist die Möglichkeit einen eigenen 
Ortsnamen zu verwenden. Durch Andreas Gregor (www.andreasgregor.de) wurde das Layout verbessert.
